# haflsl
A hobby/research project exploring compiler design and virtual machine implementation. The project includes a GLSL lexer, parser, and a compiler that transforms an AST into SPIR-V bytecode.
 
The `scripting` branch extends the project with a custom scripting language featuring OOP with inheritance, along with its own compiler and VM.
